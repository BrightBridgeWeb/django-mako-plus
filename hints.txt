These are hints to remind me (the developer) how to work with github, pypi, etc.

To upload to github, just use regular git commands.  The readme.md file will update on the site automatically.

git add --all
git commit -am 'message'
git push origin master

To upload to pypi:

- change the __init__.py file -> __version__ = 
python setup.py sdist upload
- remove the extra files/dirs it created (like dist/)
rm -rf dist/ MANIFEST



Upgrade to 3.0
  - The location of many things has refactored.  
        + Replace for all instances of "django_mako_plus.controller" with "django_mako_plus".
        + Following the Django format, dmp_render() is now the primary method for rendering to a response.
          Replace "dmp_render_to_response" with "dmp_render" and "dmp_render" with "dmp_render_to_string". 
          I suggest doing three steps:
              1. Replace "dmp_render_to_response" with "dmp_temp_render".
              2. Replace "dmp_render" with "dmp_render_to_string".
              3. Replace "dmp_temp_render" with "dmp_render".

  - Ensure "django_mako_plus.controller.router.RequestInitMiddleware" in settings.py INSTALLED_APPS was changed to 
    "django_mako_plus". 

  - Ensure "django_mako_plus.controller.router.RequestInitMiddleware" in settings.py MIDDLEWARE was changed to 
    "django_mako_plus.RequestInitMiddleware".  It will probably need adjusting.

  - Add the 'django_mako_plus.MakoTemplates' setting to the settings.py TEMPLATES.  This structure now includes
    all the DMP settings, so integrate your existing DMP settings.py information into here.  See the tutorial for
    the exact structure.
        + After you move your settings to TEMPLATES, remove the old, top-level DJANGO_MAKO_PLUS = {} from settings.py.

  - Ensure "django_mako_plus.controller.router.router_request" in urls.py was changed to 
    "django_mako_plus.route_request".  It will probably need adjusting.


To get a template using the Django API:  
  from django.template import engines
  template = engines['django_mako_plus'].get_template("homepage/index.html')
  

To compile from string using the Django API:
  from django.template import engines
  template = engines['django_mako_plus'].from_string("hello ${data}!")
  
To use the Django shortcut method:
  response = render(request, 'homepage/index.html', context={ 'var1': 'value' }, using='django_mako_plus')
